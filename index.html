Proyectos desarrollados

TelmApp | Sistema de Telemetría Apícola

Un sistema de monitoreo Apícola con app para el cliente y vista en tiempo real de la temperatura y Humedad de cada Colmena.
De Hadware se uso DHT11, Modulo ESP32.
De Software se uso Firebase para la base de datos, Kotlin para el app y Arduino (ESP32)

20 Noviembre 2020

VP4X.py | Sistema de Protección y Contraataque Cibernético

Un software de ciberseguridad que detecta y responde en tiempo real a ataques informáticos, bloqueando amenazas y registrando eventos sospechosos.
Tecnologías utilizadas:
Python para la lógica del sistema.
Scapy y PyShark para análisis de red.
SQLite para almacenamiento de logs.
IPTables y Fail2Ban para mitigación de ataques.

06 Junio 2023

Nitro Client | Contador Virtual con IA

Un sitio web que automatiza la contabilidad, permitiendo a los usuarios subir sus registros y obtener reportes financieros con inteligencia artificial.
Tecnologías utilizadas:
Django (Python) para el backend.
SQLite3 para la base de datos.
HTML, CSS y JavaScript para la interfaz.
IA integrada para análisis contable automatizado.

12 Octubre 2024

CM Client | Launcher Optimizado para BedWars

Un launcher personalizado para Minecraft Java, diseñado para mejorar el rendimiento y la experiencia en partidas de BedWars con optimización avanzada.
Tecnologías utilizadas:
Java y LWJGL para el desarrollo del cliente.
OptiFine y Sodium para mejorar FPS.
FastPackets para reducir la latencia en servidores.
Interfaz UI personalizada con JavaFX.

25 Enero 2025
